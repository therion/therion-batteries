This is a minimal TeX distribution to be used with Therion

Based on TeXLive (http://www.tug.org/tex-live/) by Sebastian Rahtz and
W32TeX (http://w32tex.org) by Akira Kakuto. The following W32TeX packages
are used:
  - lcdf-typetools-w32.tar.xz
  - pdftex-w32.tar.xz
  - web2c-lib.tar.xz
  - web2c-w32.tar.xz
  - tl-win64.zip

Upgrade to full TeXLive distribution if possible.
